# 주의: 프론트메타
jekyll은 표시할 문서에 프론트메타(front matter)라고 불리는 yaml형식의 페이지옵션을 반드시 표시해야 합니다. 아무런 설정이 필요없는 경우에도 프론트메타 구역을 표시해야 합니다.

따라서 모든 문서는 최소한 아래의 내용으로 시작해야 합니다.

```
---
---
문서 내용...
```

# 주의: 다른 형식의 문서
이 테마가 추천하는 문서양식은 마크다운(markdown)입니다. 하지만 반드시 마크다운파일로 작성할 필요는 없으며 파일 이름만 동일하다면 html이나 txt, xml도 사용할 수 있습니다. 단, 테마는 마크다운 이외의 양식을 정상적으로 출력하지 못할 수 있습니다.
